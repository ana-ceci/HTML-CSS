Teste Flexebox

Flexbox (ou Flexible Box Layout) é um método do CSS usado para organizar elementos dentro de um container, deixando o layout flexível e fácil de alinhar.

Ou seja, ele ajuda a posicionar itens na horizontal ou vertical, controlar espaçamento e alinhamento de forma muito mais simples.
